# ReverseEngineeringProtocol
Aplikácia na tvorbu protokolov pre reverzné inžinierstvo. Hodnotí náročnosť 3D skenovania (Mesh), postprocesingu a CAD modelovania. Poskytuje časový odhad a presne definuje výstupy: od STL a inšpekčných protokolov až po živé natívne dáta (Catia, NX, Solidworks). Zabezpečuje jasné zadania a efektívne nacenenie RE projektov.
